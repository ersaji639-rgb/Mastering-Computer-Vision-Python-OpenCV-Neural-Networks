# Mastering Computer Vision with Python, OpenCV, and Neural Networks

This repository contains projects, assignments, and practical implementations developed as part of the **"Mastering Computer Vision with Python, OpenCV, and Neural Networks"** program conducted by **iHUB DivyaSampark, Indian Institute of Technology Roorkee**, in collaboration with **RBPL**.

## Overview

The repository showcases various computer vision applications implemented using Python, OpenCV, and deep learning techniques. The projects cover fundamental and advanced concepts in image processing, feature extraction, object detection, image classification, and neural network-based computer vision tasks.

## Contents

* Image Processing Techniques
* OpenCV-Based Applications
* Feature Detection and Extraction
* Object Detection and Recognition
* Image Classification using Neural Networks
* Deep Learning for Computer Vision
* Practical Case Studies and Assignments

## Project Files

All project reports and demonstrations are provided in **HTML format**.

### How to View the Projects

1. Download or clone this repository.
2. Navigate to the desired project folder.
3. Open the `.html` file using any modern web browser such as:

   * Google Chrome
   * Mozilla Firefox
   * Microsoft Edge
   * Safari

No additional software installation is required to view the project reports.
